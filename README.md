# odin-landing-page (CognitoFlow Landing Page)

A modern landing page for CognitoFlow, an AI-powered data analytics platform.

## Project Overview

This landing page was created as part of The Odin Project's Foundations course. It demonstrates:

- Semantic HTML structure
- Flexbox layout techniques
- Responsive design principles
- Modern CSS styling

## Technologies Used

- HTML5
- CSS3 (Flexbox)
- Google Fonts (Roboto)

## Features

- Clean, professional design
- Four feature cards with icons
- Customer testimonial section
- Call-to-action with conversion-focused copy
- Fully responsive layout
